# Minecraft Launcher

Complete Minecraft launcher using Electron and React.

## Prerequisites
- Node.js >= 18
- npm

## Installation
```bash
git clone <repo-url>
cd minecraft-launcher
npm install
```

## Development
```bash
npm run dev
```

## Build & Package
```bash
npm run build
npm run dist
```

## Tests & Lint
```bash
npm test
npm run lint
```

See `design.md` for UI details and `CONTRIBUTING.md` for contribution guidelines.
